# Requirements

## Python

For the currently supported Python versions, please take a look the “Meta” section of the [ICOlyzer package on the Python Package Index](https://pypi.org/project/icolyzer/) (PyPI). We recommend you use a current 64-bit version of Python.

You can download Python [here](https://www.python.org/downloads). When you install the software, please do not forget to enable the checkbox **“Add Python to PATH”** in the setup window of the installer.
